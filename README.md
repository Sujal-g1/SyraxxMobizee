Mobizee is a modern, responsive smart mobility web application designed to improve urban transportation through real-time location tracking, multilingual support, digital wallet integration, green reward points, and a built-in emergency panic system.

The platform focuses on safety, accessibility, and sustainability, providing users with a seamless experience across both desktop and mobile devices.

## ✨ Key Features

- 📍 **Live Location Tracking** using GPS and reverse geocoding  
- 🌐 **Multi-language Support** (English, Hindi, Punjabi)  
- 👤 **User Account System** with dynamic profile display  
- 🚨 **Panic Button** for emergency navigation and safety  
- 💳 **Wallet Integration** for payments and balance management  
- ⭐ **Green Points Reward System** to encourage eco-friendly travel  
- 📱 **Fully Responsive UI** with smooth mobile slide-in menu  

## 🛠️ Tech Stack

- **Frontend:** React, Tailwind CSS  
- **Routing:** React Router  
- **Internationalization:** react-i18next  
- **Icons:** React Icons, Lucide React  
- **APIs:** Browser Geolocation API, OpenStreetMap Nominatim  

## 🎯 Project Goals

Mobizee aims to:

- Enhance commuter safety with quick-access emergency features  
- Promote sustainable transportation through green rewards  
- Provide a clean, scalable, and mobile-first user experience
