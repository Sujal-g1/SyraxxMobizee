# 📖 About Mobizee

Mobizee is a smart mobility and public transportation platform designed to make urban and semi-urban transportation more accessible, sustainable, and user-friendly.

The platform combines route planning, live transport information, digital ticketing concepts, multilingual accessibility, eco-friendly reward systems, emergency assistance tools, and transport authority analytics into a single ecosystem.

Mobizee was developed with the vision of encouraging people to use public transportation while improving convenience, safety, and transparency.

# 💳 Digital Wallet System

Mobizee includes a built-in wallet interface that demonstrates how digital payments can be integrated into public transportation.

Users can:

* View wallet balance
* Add funds (simulation)
* Use balance for travel operations
* Manage transportation expenses

### Important

No real payment gateway is currently integrated.

This wallet is purely a demonstration feature used to showcase future payment workflows.

The application does NOT process:

* Real payments
* UPI transactions
* Debit card payments
* Credit card payments
* Banking transactions

Future versions may include:

* UPI Integration
* Payment Gateway Support
* Recharge System
* Transaction History

# 📦 Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Additional packages used:

```bash
npm install bcryptjs

npm install qrcode.react

npm install nodemailer mongoose express dotenv

npm install canvas-confetti
```

Start the development server:

```bash
npm run dev
```
